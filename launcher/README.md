# World of Tanks 0.6.5 — Як грати

Привіт! Ця папка містить старий клієнт World of Tanks 0.6.5 та лаунчер для підключення до приватного сервера.

## Швидкий старт

1. **Перший раз** — подвійний клік на `setup.bat`
   - Windows запитає права адміністратора → натисни **Так**
   - Скрипт додасть 2 рядки у системний `hosts` файл (це налаштовує перенаправлення на приватний сервер)
   - Готово, цей крок робиться один раз

2. **Кожен раз** — подвійний клік на `play.bat`
   - Запускає гру (не вимагає прав адміна)

3. **Якщо більше не граєш** — подвійний клік на `uninstall.bat`
   - Запитає admin
   - Видаляє рядки з `hosts`, повертає все як було

## Що це за `hosts`?

`hosts` — це системний файл Windows, що каже комп'ютеру куди ходити, коли вводиш певну адресу. Ми додаємо туди 2 рядки, які перенаправляють `login-master.worldoftanks.com` на приватний сервер замість офіційних серверів Wargaming.

Це абсолютно безпечно і легко скасовується через `uninstall.bat`.

## Часті проблеми

### "Cannot find WorldOfTanks.exe"

Файли `setup.bat` / `play.bat` мають лежати **поряд** з `WorldOfTanks.exe` в папці гри. Якщо ти розпакував архів — переконайся, що нічого не загубилось.

### "Connection lost" одразу після логіну

Сервер недоступний або не запущений. Звернись до того, хто дав тобі цю гру.

### "Setup потребує адмін прав"

Так, для зміни `hosts` файлу Windows завжди потрібен адмін. Це робиться **один раз**, далі гра запускається без UAC.

### Хочу повернутись на офіційний WoT

Зараз працює тільки приватний сервер (бо `hosts` перенаправляє). Запусти `uninstall.bat` — і офіційний WoT знову запрацює (якщо встановлений). Ця версія 0.6.5 з офіційним сервером не сумісна — вона занадто стара.

## Питання?

Питай у того, хто дав тобі архів.
